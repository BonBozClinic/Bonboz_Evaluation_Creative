# Evaluation Rubric — Quick Reference

## Scoring Scale

| Score | Label | Meaning |
|-------|-------|---------|
| 10 | Exceptional | Best-in-class. Would win awards or go viral. |
| 9 | Excellent | Professional, polished, highly effective. |
| 8 | Very Good | Strong execution with minor room for improvement. |
| 7 | Good | Solid work. Above average for the category. |
| 6 | Above Average | Competent but not distinctive. Gets the job done. |
| 5 | Average | Middle of the road. Neither good nor bad. |
| 4 | Below Average | Noticeable issues that hurt effectiveness. |
| 3 | Poor | Significant problems. Likely to underperform. |
| 2 | Very Poor | Fundamental issues. Should not be published. |
| 1 | Terrible | Actively harmful to the brand. |

## Dimensions at a Glance

### 1. Hook Effectiveness (Weight: 1.5x)
Does the first 3 seconds stop the scroll?
- 9-10: Impossible to scroll past
- 5-6: Some pause, many scroll
- 1-2: No one stops

### 2. Message Clarity (Weight: 1.3x)
Can the viewer articulate the value prop?
- 9-10: Crystal clear single message
- 5-6: Present but takes effort
- 1-2: No discernible message

### 3. Visual Quality (Weight: 1.0x)
Does it look appropriate for the platform?
- 9-10: Exceptional, perfectly matched
- 5-6: Acceptable, some rough edges
- 1-2: Blurry, broken, amateur

### 4. Audience Targeting (Weight: 1.2x)
Does a specific audience feel "this is for me"?
- 9-10: Laser-targeted
- 5-6: Generic enough for anyone
- 1-2: No targeting at all

### 5. Pacing & Structure (Weight: 1.0x)
Does every second earn the next?
- 9-10: Perfect flow, builds naturally
- 5-6: Uneven, some dead spots
- 1-2: Rambling, no structure

### 6. CTA Strength (Weight: 1.3x)
Does it drive action?
- 9-10: Compelling, clear, urgent
- 5-6: Present but weak/generic
- 1-2: No CTA at all

### 7. Emotional Resonance (Weight: 1.0x)
Does it make the viewer feel something?
- 9-10: Strong emotional response
- 5-6: Informative but flat
- 1-2: Cringe or trust-destroying

### 8. Sound-Off Effectiveness (Weight: 0.7x)
Does it work without audio?
- 9-10: Fully effective muted
- 5-6: Gets the gist across
- 1-2: Completely dependent on audio

## Overall Score Formula

```
Overall = (Hook×1.5 + Message×1.3 + Visual×1.0 + Audience×1.2 +
           Pacing×1.0 + CTA×1.3 + Emotion×1.0 + SoundOff×0.7) / 9.0
```

## Quick Decision Guide

| Overall | Decision |
|---------|----------|
| 8.5+ | Ship immediately. Scale budget. |
| 7.0-8.4 | Ship with minor fixes. |
| 5.5-6.9 | Fix top 2 weakest areas first. |
| 4.0-5.4 | Major rework needed. |
| < 4.0 | Start over with new concept. |
